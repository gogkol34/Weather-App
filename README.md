# 🌦 Weather App

A simple weather application example implemented in multiple programming languages.

---

## 📖 Overview

This repository demonstrates a basic weather application written in:

- JavaScript
- Python
- C#
- PHP

The project is intended for educational purposes and can be expanded to work with real weather APIs.

---

## ✨ Features

- Display weather information
- Simple data structure examples
- Beginner-friendly code
- Multi-language implementation

---

## 📂 Project Structure

```text
javascript/
python/
csharp/
php/
```

---

## Example Output

```text
City: Kyiv
Temperature: 22°C
Condition: Sunny
```

---

## Running

### JavaScript

```bash
node weather.js
```

### Python

```bash
python weather.py
```

### C#

```bash
dotnet run
```

### PHP

```bash
php weather.php
```
